Gmail Instructions
  1. Click "Copy for Gmail"
  2. Open Gmail → click ⚙️ → See all settings
  3. Under General → Signature
  4. Create a new signature or select an existing one
  5. Delete any existing content
  6. Paste (Cmd/Ctrl + V)
  7. Set under Signature defaults
  8. Scroll down → Save Changes
Apple Mail Instructions
  1. Click "Copy for Apple Mail"
  2. Open Mail → Mail (top menu) → Settings
  3. Select your account (e.g. Google)
  4. Add a new signature (+) or select an existing one
  5. Uncheck "Always match my default message font"
  6. Delete any existing content
  7. Paste (Cmd + V)
  8. Close Settings and test in a new email
Outlook Instructions
  1. Click "Copy for Outlook"
  2. Open Outlook → Outlook (top menu) → Settings
  3. Go to Signature
  4. Create a new signature or select an existing one
  5. Click Edit
  6. Delete everything except --
  7. Paste below --
  8. Click Save
  9. Set as default for New messages and Replies/Forwards
